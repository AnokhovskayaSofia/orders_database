### Для запуска в режиме разработки
- Создайте и активируйте виртуальное окружение
```bash
python -m venv venv
source venv/Script/activate
```
 - Загрузите все пакеты и зависимости из файла requirements.txt
```bash
pip install -r requirements.txt
```
- Запустите проект
```bash
python main.py
```
